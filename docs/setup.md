# <h1 align="center">Linux</h1>

## 一、说明

### 1、setup.sh

该脚本将会卸载以下软件：

* libreoffice
* sgt-launcher
* sgt-puzzles

安装以下软件：

* git: 版本管理工具
* gcc: C语言编译器
* g++: C++编译器
* nasm: 汇编语言编译器


```bash

```

```bash
   ./xxx.sh <命令>
```
